localStorage helpers (inside the file, not exported):
```typescript
const HISTORY_KEY = (cwd: string) => `slopdock_sessions_${cwd}`;
const MAX_HISTORY = 20;
const MAX_SESSIONS = 8;

function loadHistory(cwd: string): PersistedSession[] {
  try {
    const raw = localStorage.getItem(HISTORY_KEY(cwd));
    return raw ? (JSON.parse(raw) as PersistedSession[]) : [];
  } catch { return []; }
}

function saveToHistory(cwd: string, entry: PersistedSession) {
  try {
    const existing = loadHistory(cwd);
    localStorage.setItem(HISTORY_KEY(cwd), JSON.stringify([entry, ...existing].slice(0, MAX_HISTORY)));
  } catch {}
}
```

Hook implementation: use useState for sessions array and activeId. Use useCallback for all actions. Load history from localStorage when cwd changes via useEffect.

For close(): force status to 'done' if it was 'working' (process killed mid-work) when persisting. Keep the original status if it was already terminal ('done' | 'error').
