**Component props interface:**
```typescript
interface OnboardingModalProps {
  initialPath: string | null;
  onDismiss: () => void;
}
```

**Visibility logic** (evaluate once at component mount — not on every render):
- If `initialPath !== null` → return null immediately (user has a saved folder)
- If `localStorage.getItem('slopmop_onboarded')` is set → return null (already onboarded)
- Otherwise → show the modal

Use a `useState` for visibility initialized from the above check:
```typescript
const [visible] = useState(
  () => initialPath === null && !localStorage.getItem('slopmop_onboarded')
);
if (!visible) return null;
```

**Dismiss handler:**
```typescript
function handleDismiss() {
  localStorage.setItem('slopmop_onboarded', '1');
  onDismiss();
}
```
